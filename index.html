Kenzo's aimbot calculator

This site is a tool for calculating the view angle in degrees for an aimbot. On the tool itself you have to enter the: x and y position for the enemy, player and the player view orientation (angle) in degrees. With these values the tool is able to calculate the angle you should look to hit the enemy. North is 0 degrees, South is 180 degrees.
